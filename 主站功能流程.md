# 主站正常流程

1. **扫描从站**
   - 主站识别总线上的所有从站。
   - 读取每个从站的 SII/EEPROM 信息，并据此配置邮箱对应的 Sync Manager 0/1。

2. **进入 Maintenance（PREOP）**
   - 所有从站进入 PREOP 状态，mailbox 建立。
   - 主站为每个从站建立节点占位，SDO 通信就绪。

3. **SDO 维护**
   - 执行从站诊断、参数读写、状态巡检等维护操作。

4. **进入 Operational（可选）**
   - 配置 PDO 映射、SafeOp、DC 同步。
   - 所有从站进入 OPERATIONAL，开始周期性 PDO 数据交换。

5. **周期运行与监控**
   - 每个周期执行 PDO 收发。
   - 持续监控从站状态，异常时进入 Error 状态并停机。
