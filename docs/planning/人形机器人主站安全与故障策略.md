# 人形机器人主站安全与故障策略

## 1. 文档定位

本文档细化《人形机器人 EtherCAT 主站功能规划》中的安全与故障处理部分，用于定义主站面对通信异常、伺服异常、上层命令异常和外部急停时的分级响应策略。

本文档只定义主站侧的软件策略和外部安全接口，不替代功能安全认证设计。

## 2. 故障分级

| 等级 | 含义 | 典型场景 | 默认响应 |
|------|------|----------|----------|
| Normal | 正常运行 | WKC、DC、伺服状态均正常 | 持续周期控制 |
| Warning | 可继续运行但需要提示 | 周期 jitter 接近阈值、温度接近阈值、非关键诊断失败 | 记录日志并上报 |
| RecoverableFault | 可恢复故障 | 单轴伺服 Fault、SDO 临时失败、非关键轴组异常 | 停止相关轴或轴组，允许诊断和复位 |
| FatalFault | 不可继续运行 | 关键从站掉线、连续 WKC 异常、DC 同步丢失、配置不匹配 | 全身 Quick Stop 或进入 Error |
| Emergency | 紧急安全事件 | 外部急停、硬限位触发、严重跟随误差、上层命令长时间丢失 | 最高优先级停机，禁止自动恢复 |

## 3. 停机策略

| 策略 | 用途 | 要求 |
|------|------|------|
| Hold | 短时间命令延迟或轻微异常 | 保持目标值前必须确认伺服模式和驱动器行为 |
| Controlled Decel | 命令超时、可控降级 | 按配置减速度下发目标或速度 |
| Quick Stop | 通信仍可用时的主要安全停机 | 通过 `0x6040` 触发，监测状态变化和超时 |
| Disable Operation | 轴组不允许继续输出力矩时 | 确认抱闸和机械状态后执行 |
| Disable Voltage | 严重故障或停机失败时 | 可能导致失力，必须结合抱闸/STO 策略 |
| STO | 外部安全链路断开动力输出 | 主站负责观测和记录，不应替代安全硬件 |

## 4. 故障处理矩阵

| 故障 | 分级 | 检测方式 | 推荐响应 | 恢复条件 |
|------|------|----------|----------|----------|
| 单周期 WKC 异常 | Warning | PDO 周期检查 | 记录并计数 | 后续周期恢复 |
| 连续 WKC 异常 | FatalFault | 连续 N 周期异常 | 全身 Quick Stop / Error | 重新扫描或确认链路恢复 |
| 从站掉线 | FatalFault | 状态监控、WKC、拓扑变化 | 全身 Quick Stop / Error | Stop 后重新初始化 |
| DC 偏差超限 | Warning / FatalFault | DC 监控 | 告警或停机，按阈值配置 | 偏差回到窗口并人工确认 |
| SDO 超时 | RecoverableFault | SDO 返回值和超时 | 当前 Activity 失败 | 重试成功或重新进入维护 |
| 伺服 Fault | RecoverableFault | `0x6041` / `0x603F` | 停止相关轴或轴组 | 故障原因清除后 Fault Reset |
| 跟随误差超限 | Emergency | 驱动器错误码或主站监测 | 相关轴组或全身急停 | 人工确认机械安全 |
| 软限位触发 | RecoverableFault | 主站限位检查 | 拦截命令或停止轴 | 目标回到合法范围 |
| 硬限位触发 | Emergency | IO/驱动器状态 | 最高优先级停机 | 人工确认并复位 |
| 上层命令超时 | FatalFault / Emergency | 周期序号或时间戳 | Controlled Decel 或 Quick Stop | 上层恢复并人工确认 |
| 外部急停 | Emergency | 安全 IO/STO/FSoE 状态 | 立即停机并禁止自动恢复 | 急停解除并人工确认 |

## 5. 恢复原则

1. Emergency 不允许自动恢复，必须人工确认。
2. FatalFault 默认需要 Stop 后重新初始化，除非有明确安全论证。
3. RecoverableFault 可以支持轴或轴组级恢复，但必须保留故障快照。
4. Fault Reset 前必须确认故障原因已消除。
5. 抱闸释放前必须确认轴已使能、目标稳定、无急停、无硬限位。
6. 恢复流程必须记录每一步状态变化和失败原因。

## 6. 验收要求

每类 M 级故障至少需要：

- 一条故障注入用例。
- 明确的触发条件和超时阈值。
- 停机策略验证结果。
- 故障快照样例。
- 恢复流程验证结果。
