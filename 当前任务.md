## 验收标准

1. `ControllerState` 枚举只有 6 个值。
2. `StartOperation()` 内部仍然顺序执行 PDO 配置、SafeOp、DC 配置、Op，但不更新中间状态。
3. `CheckSlaveStates()` 只检查 `kMaintenance`（期望 PREOP）和 `kOperational`（期望 OP）。
4. `EcatApplication::Run()` 的 switch-case 对应 6 个状态。
5. 所有命令按新状态过滤：
   - `kAdapterReady`：`scan` / `stop` / `help` / `exit`
   - `kScanned`：`config` / `stop` / `help` / `exit`
   - `kMaintenance`：`diagnose` / `param` / `inspect` / `start` / `stop` / `help` / `exit`
   - `kOperational`：`stop` / `exit`
   - `kError`：`stop` / `exit`
6. 编译通过，`ecat_scan` 仍能让从站进入 PREOP 并执行 SDO 诊断。
7. 不进入 SAFEOP/OP，不启动独立 PDO 周期线程。
