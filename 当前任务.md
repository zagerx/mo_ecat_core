# 当前任务

## 目标

搭建健壮的 EtherCAT 主站初始化流程，确保所有从站能稳定进入 **PREOP** 状态，并在此状态下支持 **SDO 通信**。

**当前阶段不涉及 PDO 通信。** PDO 映射、IOmap 配置、DC 同步、进入 OP 等放到后续阶段再实现。

## 当前不做的事情

| 事项 | 原因 |
|---|---|
| PDO 映射配置 | 当前目标是 PREOP + SDO，PDO 放到后续 |
| `ConfigureProcessData()` | 会进入 SAFEOP 并配置 IOmap，当前不需要 |
| `ConfigureDc()` | DC 同步只在 PDO 周期运行时才需要 |
| `StartOperation()` | 会请求 OP 并启动 PDO 通信，当前不需要 |
| PDO 周期线程 | 不建立 PDO 通信，无需周期线程 |

---

## 验收标准

1. 运行 `ecat_scan` 后，所有从站成功进入 `PREOP` 状态。
2. `Initialize()` 返回 `true`。
3. 能通过 SDO 成功读取从站对象字典（如 `0x1000`、`0x1018`）。
4. 状态转换失败时，程序能正确报错并回滚到 `INIT` / 关闭 SOEM。
5. 程序不尝试进入 `SAFEOP` 或 `OP`。
