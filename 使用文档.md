# MoDriverPC_EtherCAT 使用文档

## 一、获取代码并初始化子模块

```bash
git submodule update --init --recursive
```

## 二、编译

```bash
cmake -B build .
make -C build -j$(nproc)
```

## 三、运行

### 查看可用网卡

```bash
sudo ./build/ecat_scan
```

### 扫描并测试 EtherCAT 从站

```bash
sudo ./build/ecat_scan enp0s31f6
```

> 把 `enp0s31f6` 替换成实际的网卡名。
